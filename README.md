# Daily AI News

每日 AI 新闻简报技能 - 从多个来源聚合并总结最新的 AI 新闻，提供简洁的新闻摘要并附带原文链接。

## 功能特点

- **多源聚合**：从 VentureBeat、TechCrunch、The Verge 等主流 AI 新闻网站获取内容
- **智能筛选**：自动去重，保留最近 24-48 小时的重要新闻
- **分类整理**：按 5 个类别组织新闻（重大公告、研究论文、行业动态、工具应用、政策伦理）
- **简洁摘要**：提供要点式总结，附带原文链接

## 何时使用

当用户出现以下情况时激活此技能：

- 询问今天的 AI 新闻或最新 AI 发展
- 请求每日 AI 简报或更新
- 想知道 AI 领域正在发生什么
- 询问 AI 行业新闻、趋势或突破

## 使用示例

```
给我今天的 AI 资讯
AI 有什么新动态？
最近有什么重要的 AI 研究论文？
```

## 工作流程

```
阶段 1：信息收集
  ├─ 直接网站抓取（3-5 个主要 AI 新闻网站）
  └─ 带日期过滤的网络搜索
      ↓
阶段 2：内容筛选
  ├─ 保留：最近 24-48 小时、重大公告
  └─ 移除：重复内容、次要更新、旧内容
      ↓
阶段 3：分类
  └─ 组织成 5 个类别
      ↓
阶段 4：输出格式化
  └─ 呈现链接和结构
```

## 新闻类别

| 类别 | 内容 |
|------|------|
| 🔥 重大公告 | 产品发布、模型更新、公司公告 |
| 🔬 研究与论文 | 学术突破、研究论文、新技术 |
| 💰 行业与商业 | 融资、并购、合作伙伴关系 |
| 🛠️ 工具与应用 | 新 AI 工具、开源框架、开发者资源 |
| 🌍 政策与伦理 | AI 法规、安全讨论、社会影响 |

## 自定义选项

- **关注领域**：仅研究论文、产品发布、行业新闻、特定公司
- **深度级别**：简要、标准、深入
- **时间范围**：最近 24 小时、3 天、一周
- **格式偏好**：按类别、按时间顺序、按公司
