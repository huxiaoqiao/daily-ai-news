# 输出格式模板

用于不同输出格式和详细程度级别的预定义模板。

## 标准格式（默认）

## 模板选择指南

| 格式 | 使用场景 | 长度 | 详细程度 |
|--------|----------|--------|--------------|
| **标准** | 默认选择，每日简报 | 中等 | 摘要 + 要点 |
| **简要** | 快速浏览，时间有限 | 短 | 仅标题 |
| **深度** | 研究，深入理解 | 长 | 分析 + 影响 |
| **按时间顺序** | 跟踪随时间的发展 | 中等 | 按时间组织 |
| **按公司** | 跟踪特定公司 | 中等 | 按公司组织 |
| **仅研究** | 学术关注，研究人员 | 中等 | 论文关注 |

---

## 自定义指南

### 调整模板

1. **混合搭配**：组合不同模板的元素
2. **调整深度**：根据用户偏好添加/删除部分
3. **灵活分类**：如果需要，使用不同的类别名称
4. **本地化内容**：为不同语言翻译类别名称

### 模板元素

这些元素可以添加到任何模板中：

**部分：快速统计**
```markdown
## 📊 快速统计

- **总故事数**：[X]
- **最活跃的公司**：[列表]
- **热门主题**：[列表]
- **情绪**：[积极/中性/混合]
```

**部分：阅读时间**
```markdown
**预计阅读时间**：[X] 分钟
```

**部分：相关故事**
```markdown
**相关**：[相关故事链接]
```

**部分：标签/关键词**
```markdown
**标签**：[#标签1] [#标签2] [#标签3]
```
